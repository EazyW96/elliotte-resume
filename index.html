UBID Online Auction System

Course: SWE 6633 – Software Project Planning & Management (Summer 2024)

Team Members: Hunter Blake, Mekonnen Kindo, Connor Bland, Elliotte Wideman

UBID is an online auction system designed to help a traditional on-site auction business expand to the web. We implemented bidding, item listings, user registration, and backend integration using web development technologies.
